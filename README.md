<p>Simple Task App on pure js</p>
<p>Features:
    <ul>
        <li>Add/remove task</li>
        <li>Remove all tasks</li>
        <li>Sort tasks by adding order (id)</li>
        <li>Sort by name</li>
        <li>30 tasks maximum are allowed</li>
        <li>Local Storage is using to store tasks</li>
    </ul>
</p>
